The goal of this study is to rate the quality of explanations provided for the decision of a unknown model classifing images.

For each example, you will be presented with an image, the model's predicted class and four explanations for the model's prediction, each extracted with a different method. Each explanation consists of an heatmap with the same size as the image, where the color around each pixel indicating the importance of the pixel for the model's prediction.

You will be asked to rank the quality of the four different explanation for each image, according to how much you trust the explanation to represent relative importance of the pixels for the model's prediction. Please use numerical rankings, from 1 to 4, where 1 and 4 means that you trust the explanation the most and least, respectively. Ties are allows ("1, 1, 3, 4" is a valid ranking).
